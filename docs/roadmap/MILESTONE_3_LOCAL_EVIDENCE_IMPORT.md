# Milestone 3 - Local Evidence Import v0.1

## Goal

Allow CustosOps to analyze real endpoint evidence generated by the local read-only collector.

Before this milestone, CustosOps could show sample endpoint findings.
After this milestone, the operator can select a local endpoint evidence JSON file and send it to the local backend for analysis.

## Workflow

1. Run the endpoint collector:

   collectors/powershell/endpoint/Get-EndpointSecurityEvidence.ps1 -OutputPath reports/endpoint-evidence.local.json

2. Start the backend:

   scripts/run-backend.ps1

3. Start the frontend:

   scripts/run-frontend.ps1

4. Open:

   http://localhost:5173

5. Use "Select endpoint evidence JSON".

6. Select:

   reports/endpoint-evidence.local.json

7. Review imported endpoint findings.

## Safety

The import workflow does not upload evidence to the cloud.

The browser sends the selected JSON file only to the local backend at:

http://127.0.0.1:8000/api/endpoint/analyze

## Current Limitations

- Import is endpoint-only.
- Evidence is analyzed in browser session and backend response only.
- No persistent local history yet.
- No generated HTML report yet.
- No PDF export yet.
