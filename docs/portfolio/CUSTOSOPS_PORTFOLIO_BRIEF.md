# CustosOps Portfolio Brief

## One-line description

CustosOps is a local-first cybersecurity evidence, posture, and reporting platform for endpoint, DNS, application log, Windows Event, and IIS/Application evidence.

## What it demonstrates

```text
- Practical cybersecurity evidence handling.
- Local-first security tooling.
- FastAPI backend design.
- React/TypeScript frontend design.
- Report generation and archive workflows.
- Redaction-safe output handling.
- Run history and evidence traceability.
- Windows/IIS/Application support awareness.
- Professional dashboard UI.
- Beginner-friendly launch packaging.
- Release discipline with repeatable proof gates.
```

## Why it matters

CustosOps focuses on a real operational problem: turning local evidence into findings, reports, run history, and reviewable proof without uploading sensitive data to a cloud service.

## Demo story

```text
1. Launch with LAUNCH_CUSTOSOPS.bat.
2. Open Overview and explain local-first evidence handling.
3. Import or collect sample evidence.
4. Show findings and severity handling.
5. Generate a report.
6. Open Archive and Run History.
7. Show Redaction Settings and explain public-safe reporting.
8. Mention Desktop UI proof and contract audits as release gates.
```
